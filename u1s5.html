1.5 Order in Calculations/Evaluating Formulas

Simplify each expression. Show each step one at a time.

  1. \(20(3+7)\)
  2. \(12+4^{2}\)

    The calculator turns off.

  3. \(6(5-3)^{2}\)

    The print gets darker.

  4. \(200 \times(1+0.04 / 12)^{5 \times 12}\)

    The print get lighter.

  5. \(50 \times \frac{\left[(1+0.05 / 12)^{12 \times 10}-1\right]}{(0.05 / 12)}\)

    When using functions above the keys.

  6. \(\frac{180,000 \times(0.065 / 12)}{\left[1-(1+0.065 / 12)^{(-12 \times 15)}\right]}\)

    Clear

  7. \(\pi(8)^{2}\)

    No

  8. \(\frac{4}{3} \pi(7.5)^{3}\)

    Returns to the home screen

  9. \(\frac{1}{2} \times 15(3.2+10.8)\)

    \(-37\)

  10. Evaluate \(I=\operatorname{Prt} \text { when } P=2000, r=0.045, \text { and } t=\frac{11}{12}\)

    The previous problem reappears.

  11. Find the simple interest earned on $3000 at an annual interest rate of 3.75% for 4 years. Use the simple interest formula \(I=Prt\) where \(P\) is the principle invested, \(r\) is the annual interest rate in decimal form, and \(t\) is the number of years.

    The + sign is changed to a -.

  12. Evaluate \(A=P\left(1+\frac{r}{n}\right)^{n t} \text { when } P=2000, r=0.045, n=12, \text { and } t=\frac{11}{12}\)

    \(33\)

  13. Find the accumulated amount when $3000 is invested for 4 years at a 3.75% interest rate compounded monthly. Use the compound interest formula \(A=P\left(1+\frac{r}{n}\right)^{n t}\) where \(P\) is the principle invested, \(r\) is the annual interest rate in decimal form, \(t\) is the number of years, and \(n\) is the number of compounding periods in a year.

    \(.5\overset-3\)

  14. Evaluate \(A=P M T \times \frac{\left[\left(1+\frac{R}{n}\right)^{n t}-1\right]}{\left(\frac{R}{n}\right)} \text { when } P M T=\$ 250, R=0.02, t=20, \text { and } n=12\)

    \(\frac8{15}\)

  15. Find the accumulated amount when $1000 is invested every month for 3.5 years at a 3% interest rate compounded monthly. Use the savings plan formula \(A=P M T \times \frac{\left[\left(1+\frac{R}{n}\right)^{n t}-1\right]}{\left(\frac{R}{n}\right)}\) where \(PMT\) is the monthly payment, \(R\) is the annual percentage rate, \(t\) is the number of years, and \(n\) is the number of compounding periods per year.

    \(\frac34\)

  16. Evaluate \(a^{2}+b^{2}=c^{2} \text { when } a=5 \text { and } b=12\)

    \(\frac1{25}\)

  17. Find the length of the hypotenuse of a right triangle when the legs are 14 cm and 48 cm. Use the Pythagorean Theorem \(a^{2}+b^{2}=c^{2}\).

    \(\frac{19}{29}\)

  18. Evaluate \(x=\frac{-b \pm \sqrt{b^{2}-4 a c}}{2 a}\) when a = 3, b = -6, and c = 2. Find both solutions to the nearest hundredth.

    \(252\)

  19. Find the solutions to the equation \(-2 x^{2}+12 x+10=0\). The equation is in standard form \(a x^{2}+b x+c=0\). Use the quadratic formula \(x=\frac{-b \pm \sqrt{b^{2}-4 a c}}{2 a}\) to solve for x. Find both solutions to the nearest hundredth.

    \(\frac{32}{8}=4\)
